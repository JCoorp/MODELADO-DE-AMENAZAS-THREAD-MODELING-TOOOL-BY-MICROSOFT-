# Proyecto: Reporte HTML — Modelado de Amenazas (STRIDE + DREAD) [NEON]

## Abrir localmente
1) Abre `index.html` en Chrome/Edge/Firefox.
2) Para PDF (A4): Imprimir → Guardar como PDF.

## Entrega
- Si entregas HTML: comprime la carpeta completa.
- Si entregas PDF: imprime el HTML a PDF (A4).

## Hosting (opcional, por si lo quieres subir)
### GitHub Pages (rápido)
1) Crea un repo y sube la carpeta completa.
2) Settings → Pages → Deploy from branch → main / root.
3) Te dará una URL pública del reporte.

## Estructura
- index.html
- assets/css/styles.css
